Projects Overview

Projects

Welcome! This portfolio includes some of the personal projects I have worked on in class and in my spare time. My main areas of interests include natural language processing, computer vision, and other fun things.
To see the projects for my internships, you can look at my resume or in the about me section!

Question Answering System

A system that generates relevant and grammatically sound questions from Wikipedia articles. It also provides concise and correct answers to questions about the text.

HKIME

Open sourced intelligent input method for Cantonese Jyutping. Features include treatment of fuzzy (ambiguous) Jyutping, word prediction, and typo correction.

Computer Systems Labs

  • Malloc: Segregated explicit lists implementation of a dynamic memory allocator (malloc) in C optimized for throughput and utility.
  • Shell: Built a Linux shell program that supports job control and I/O redirection with knowledge of process control and signalling.
  • Proxy: Concurrent web proxy with cached web objects
  • Cache: Simulated memory cache with hit/miss/evict behavior, optimized clock cycles of hit/miss for doing a matrix transpose

MovieAssist

A system that utilizes openCV, EAST text detector and tesseract OCR to recognize a movie from its poster. There is also a tkinter GUI that allows interaction with a recommendation system using the MovieLens dataset.

Drum.AR

openCV based drumset that was built for Hack112, a hackathon at Carnegie Mellon University. We won second-place for the project.

Raspberry Pi Alexa

A fully functional Alexa running on a Raspberry Pi with a customizable wake word.
